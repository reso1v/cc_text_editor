# ChatLog Magician (Инструмент для редактирования текста под CC. Версия для русскоязычного проекта GTA World)

## Обновление v3.0

- Полностью изменен внешний вид инструмента: поле ввода текста и предпросмотр изображения теперь поделены на две части, что сделало редактирование контента более удобным;
- Исправлены ошибки при изменении межстрочного интервала: раньше инструмент оставлял системный отступ даже при 0.1, однако сейчас ошибка исправлена и добавлена возможность менять межстрочный интервал более динамично;
- Прекращена поддержка системы обучения использования инструментом: вы написали, что обучение крайне странно реагировало на мониторах 1080p, поэтому обучение полностью вырезано и возможно будет возвращено в дальнейшем обновлении;
- Временно прекращена поддержка установки собственного цвета для строк: постараюсь вернуть возможность изменения цвета в ближайшее время;
- Полностью исправлена логика сохранения изображения
